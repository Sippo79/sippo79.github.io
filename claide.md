# Project Rules

## 開発方針

- GitHub Pages で公開する静的サイト
- HTML/CSS/JavaScript構成
- ビルドツール無し
- SEO重視
- モバイルファースト
- PWA対応
- ダーク背景 + 青紫ネオン + ガラスUI

## 作業ルール

- HTML/CSS/JS/JSON修正は確認不要
- PC環境へ影響する操作のみ確認
- npm install
- pip install
- shell command
- git reset
- VSCode拡張変更
は必ず事前確認

## デザインルール

- 高級感
- 近未来感
- シンプル
- 初心者向け
- スマホ見やすさ重視

## SEOルール

- title最適化
- meta description必須
- OGP対応
- canonical対応
- sitemap更新
- semantic HTML推奨

## コードルール

- 可読性重視
- 共通化優先
- 重複回避
- 既存デザインを壊さない
- 不要なライブラリ追加禁止
