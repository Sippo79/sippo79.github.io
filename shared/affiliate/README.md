# アフィリエイトURL管理ファイル

## 目的

`gpu-guide`、`game-pc-guide`、`pc-build-check` の各サイトで共通利用するアフィリエイトURLを一元管理するファイルです。
URLの追加・変更はこのファイルのみを編集することで、全サイトへ反映できます。

---

## カテゴリ構成

| カテゴリキー  | 内容                         |
|-------------|------------------------------|
| `gpus`      | グラフィックボード単体           |
| `bto`       | BTOゲーミングPC（各ショップ別）   |
| `monitors`  | ゲーミングモニター               |
| `accessories` | 周辺機器（アケコン・マウスなど）  |

---

## URLを追加するときのルール

1. **既存キーへの追記**：対象の商品キー（例: `rtx5070`）を探し、該当モールのURLを空文字 `""` から実際のURLに置き換える。
2. **新しい商品を追加する場合**：既存エントリを参考に同じ構造でオブジェクトを追加し、`status` を `"preparing"` で登録する。
3. **URLはアフィリエイトタグ付きの最終URLを記載**する（短縮URL・リダイレクトURLは避ける）。
4. **キー名はキャメルケース**（例: `gamingMonitor144hz`）で統一する。

---

## status の使い方

| 値           | 意味                                   |
|-------------|----------------------------------------|
| `preparing` | URLが未設定・審査中など、まだ使えない状態  |
| `active`    | URLが設定済みで、サイトに表示してよい状態  |
| `paused`    | 一時的に非表示にしたい場合               |
| `discontinued` | 商品・サービスが終了し、今後使わない状態 |

サイト側では `status === "active"` のエントリのみ表示するよう実装する想定です。

---

## 各サイトからの参照について

各サイト（`gpu-guide` など）は、ビルド時またはランタイムにこのファイルを読み込んで使用します。
URLを差し替えたい場合は **このファイルだけ編集すれば全サイトに反映**されます。

各サイト側の読み込み実装は別途対応予定です。
